PROJECT: Multilinked Structures (.prd / .prs)

GOAL
-----
Реализовать систему хранения многосвязной структуры:
- Список компонентов (.prd)
- Список спецификаций (.prs)
- Консольное приложение (задание 1)
- Оконное приложение (задание 2)

ARCHITECTURE
------------
Слои:
1. domain          — сущности и правила
2. application     — use cases (команды)
3. infrastructure  — бинарные файлы и репозитории
4. adapters        — CLI / GUI

Ни один слой не должен зависеть от внешнего (зависимости направлены внутрь).

IMPORTANT RULES
---------------

- Деталь не может иметь спецификацию
- Нельзя удалить компонент, если на него есть ссылки
- Удаление логическое (бит = -1)
- Truncate делает физическую очистку
- Все указатели — offset от начала файла
- Значение -1 = null pointer
