### Работа с Socet
#### Реализация двух приемников на одном компьютере: Client и Server.
Клиент должен подключаться по адресу к серверу, далее иметь возможность отправлять
либо название директории либо название  txt файла. 

Сервер в свою очередь должен ожидать подключение клиента и принимать на вход название файла или дирректории и отправлять либо содержимое файла либо список элементов в дирректории.
