<task type="auto">
  <name>Task 2: Convert /// comments to proper JavaDoc format</name>
  <files>

    - L4S4-Maven/src/main/java/base/Client.java
    - L4S4-Maven/src/main/java/SelectTaskMenu.java
    - L4S4-Maven/src/main/java/task1/FileServer.java
  </files>
  <action>
Convert all `///` comments to proper JavaDoc `/** */` format:

Before:
```java
/// Шаблон сервера
public class Server {
```

After:
```java
/**
 * Шаблон сервера для клиент-серверного взаимодействия.
 * Предоставляет базовую функциональность для создания сокетных серверов.
 */
public class Server {
```

For methods, ensure JavaDoc includes @param and @return:
```java
/**
 * Отправить сообщение клиенту и получить ответ.
 *
 * @param str сообщение для отправки
 * @return ответ от клиента
 * @throws IOException при ошибке ввода-вывода
 */
public String write(String str) throws IOException {
```

Apply this to all public classes and public methods in:
- base/Server.java
- base/Client.java
- SelectTaskMenu.java
- task1/FileServer.java
- task1/ClientRequest.java
- task1/Cortege.java
  </action>
  <verify>
    <automated>grep -r "^///" L4S4-Maven/src/main/java/ 2>/dev/null | wc -l</automated>
    Expected: 0 (no /// comments remaining)
  </verify>
  <done>All /// comments converted to proper JavaDoc format.</done>
</task>
