# Проект: L4S4-Maven

**Дата создания:** 2026-04-06
**Статус:** Brownfield (существующий проект)
**Тип:** Учебный проект по сетевому программированию

---

## Описание проекта

Учебный проект по работе с Java Socket API. Реализует три задачи клиент-серверного взаимодействия:

1. **Task1** — Файловый сервер: просмотр директорий и чтение файлов через сокеты
2. **Task2** — Контроллер технологического процесса: генерация и передача данных температуры/давления
3. **Task3** — Система мониторинга установок: визуализация состояний промышленных установок
